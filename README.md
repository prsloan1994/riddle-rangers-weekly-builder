# Riddle Rangers Weekly Builder

A classroom website for making and presenting a four-week run of Riddle Rangers puzzles.

Open `index.html` in a browser, or serve the folder locally. The app saves the four-week program in the browser and includes export/import buttons for moving the program to another computer.

## Included puzzle types

- Top-half letters: students see only the tops of the answer letters.
- Fallen letters: selected letters are removed from the answer and shown in a letter bank.
- Backwards words: students reverse the printed words.
- Square letters: students keep only letters shown in squares.
- Symbol key: students replace symbols with matching letters.
- Next-letter code: students move each coded letter to the next alphabet letter.
- Bible lookup: students compare verses to find the matching book.
- Cross-out letters: students remove extra letters from a phrase.

## Presenting

Use **Present Week** for the front-of-room mode. Space or Right Arrow moves through clue, puzzle, hint, and reveal. Up/Down move between riddles.

The presenter also has live-solving controls for class participation: type the kids' current guess, reveal one answer letter at a time, show the hint, reveal the full answer, or reset the solve.

Use the **Program week** selector to switch between Week 1, Week 2, Week 3, and Week 4. Each week keeps its own title, theme, audience, and riddles.
